Python Quiz Game

This is a simple, beginner-friendly Python quiz game that uses basic OOP concepts.
The game asks multiple questions, checks your answers, and keeps track of your score.
It’s a fun way to learn Python classes, objects, attributes, and methods.

Features
Uses basic OOP with Question and QuizBrain classe
True/False questions
Automatic score tracking
Beginner-friendly code
Runs entirely in the terminal

How It Uses OOP
Question class → stores each question and its correct answer
QuizBrain class → manages quiz flow, asks questions, checks answers, tracks score
Objects → each question is a Question object; quiz is a QuizBrain object
Methods → next_question(), still_has_question() control the quiz
Attributes → score, question_number, question_list
